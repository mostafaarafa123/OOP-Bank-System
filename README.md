#  OOP Bank Management System

A console-based banking application that demonstrates Object-Oriented Programming (OOP) principles and persistent data storage using SQLite.

##  Features
- **Account Types:** Supports `SavingsAccount` and `CurrentAccount` with specific business logic (e.g., overdraft limits).
- **Security:** Integrated login system for users and administrative access for viewing overall system activity.
- **Persistence:** All data (accounts and transactions) is stored in a local `bank.db` SQLite database.
- **Transaction History:** Automated logging and retrieval of all financial transactions.

##  Tech Stack
- **Language:** Python
- **Database:** SQLite3
- **Design Pattern:** Object-Oriented Programming
